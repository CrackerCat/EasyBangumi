# 纯纯看番
<img src="./easy.png" width="30%" />
一个追番和看番的 Android 应用

# 试用
`app/release` 中有安装包，可以直接安装体验

# 功能
- 番剧搜索
- 首页推荐
- 番剧收藏（追番）
- 番剧播放
- 进度记忆
- 日夜间模式切换

# 未来展望
- 自定义番剧源（初步设想用 JS 定义）
- 播放器优化（支持截图等细节）
- 番剧时刻表

# 目前官方支持番剧源
- 樱花动漫 (http://www.yhdm.so)
- BimiBimi (http://bimiacg2.net)
